How to use?

  1. Link animate.style css in your document
  2. animate.style cdn example: <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css" />
  3. Link zems_animate.js to your Document
  4. Add class="animate__animated" in any selector to initialize the animation
  5. Add custom attribute to that selector data-zems="animate__fadeInDown"
  6. Go to animate.style copy any animation style from there and place inside custom attribute data-zems="your-any-style-from-animate.style"
  7. Congrats!!: Your parallax effect working now
